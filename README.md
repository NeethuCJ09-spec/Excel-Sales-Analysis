AdventureWorks Sales Analysis using Excel

Project Overview

This project analyzes the AdventureWorks sales dataset using Microsoft Excel. The analysis focuses on understanding sales performance, profit trends, and order quantity across different regions and categories.

Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Analysis

Dataset

The dataset used in this project contains sales data from the AdventureWorks dataset, including information about regions, revenue, profit, cost, and order quantities.

Analysis Performed

- Revenue and Profit analysis by region
- Profit percentage calculation
- Yearly order quantity trend
- Category-wise sales performance
- Data visualization using charts

Key Insights

- Certain regions generated significantly higher revenue and profit.
- Profit percentage remains consistent across several regions.
- Sales trends vary across different time periods.
- Visual charts help identify patterns in order quantity and revenue.

Project Files

- Sales_Analysis.xlsx – Excel file containing pivot tables, charts, and analysis.
- Screenshots – Images showing the analysis dashboard and charts.

Purpose of the Project

This project was completed as part of a Data Analytics learning exercise to practice data analysis and visualization using Excel.
